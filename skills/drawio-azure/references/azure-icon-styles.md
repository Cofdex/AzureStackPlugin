# Azure Icon Styles — draw.io Reference

Base URL: `https://code.benco.io/icon-collection/azure-icons/`

Every icon is embedded in draw.io using this style pattern:
```
shape=image;aspect=fixed;verticalLabelPosition=bottom;verticalAlign=top;imageAspect=0;image=https://code.benco.io/icon-collection/azure-icons/<Filename>.svg;
```

## Icon Search Tips

If you can't find an icon in this catalog:

1. Try the **generic** icon: `Cognitive-Services.svg` for any AI service, `Storage-Accounts.svg` for any storage, `Monitor.svg` for any observability tool
2. Use the **closest match** — e.g., `Function-Apps.svg` works for both Durable Functions and regular Functions
3. For brand-new services not yet in the icon set, use a **text label in a rounded rectangle** instead of a broken image link
4. The full icon gallery is browsable at: https://code.benco.io/icon-collection/azure-icons/
